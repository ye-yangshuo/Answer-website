# 行测网站



## 项目目标:

搭建一个能够生成（题目——答题——答案解析）的网站



## 项目流程与进度：

1. 获取行测题目源数据

   进度：已完成，获取了常识判断题40000道

2. 通过智谱ai的api将源数据转换成（题目-选项-答案解析）格式

   进度：完成2800题目转换

3. 搭建数据库，将转换后的数据录入数据库

   进度：完成2800道录入

4. 搭建flask后端，读取数据库数据

   进度：正在进行

5. 搭建vue前端，请求后端

   进度：正在进行

6. 美化前端界面

   进度：正在进行

7. ai创建题目

   进度：构想中
